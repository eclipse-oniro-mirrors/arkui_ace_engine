## 测试样例

### List滚动退后台测试
验证List中的图片在退后台后能否正常恢复显示

### 多种资源类型退后台恢复测试
验证Resource、PixelMap、Base64、Network四种资源类型在退后台后能否正常恢复显示

### Tab切换退后台恢复测试
验证Tab场景中，退后台后图片能否正常恢复显示

### 条件控制图片显示
验证条件控制图片显示/隐藏时的内存回收

### 嵌套滚动场景
验证嵌套滚动时的图片回收

### 自定义组件中的图片
验证自定义组件中的图片在退后台后能否正常恢复显示

### 综合场景测试
验证真实应用场景（多资源类型+页面切换）的退后台恢复显示

### 快速切换场景
验证快速切换时的图片回收

### 嵌套组件场景
验证嵌套组件中图片的回收

### 图片恢复显示测试

---

## 重要说明

**回收触发条件：只有退后台才会触发回收**
- 滚动、Tab切换、if/else切换等操作不会触发回收

**核心目标：不能白屏！**
- 退后台后内存下降 ✅
- 返回前台图片正常显示 ✅
- 没有白屏或占位图 ✅

---

## 测试步骤

1. 运行样例
2. 开启"内存回收"开关
3. 上划退后台
4. 返回前台
5. 验证：图片是否正常显示？
